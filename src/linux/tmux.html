tmux

tmux is a terminal mux-er - it's basically here to make your life easier, here is my cheatsheet

Installing tmux

Installing tmux on your favourite distro is straight forward as it's typically included in the default packages:

apt-get install tmux
or
yum install tmux

Attaching\Detaching and the control command

tmux to create a new session
or
tmux a to reattach a session

Within tmux you will need to use a command to access the 'menu' by default this is Ctrl + b - you're all set, now you just need to remember your favourite commands.

Commands

##Splitting Screens into panes
" to split the screen vertically
or
% for horizontally

##Changing panes
Arrows to move between panes

##Windows
c to create a new window
n for next window
& to close window
w list windows
0 ... 9 select window by list number

##Searching and Scrolling
[ to enter copy mode
or
PgUp to scroll up


##tmux help
? to show the help menu

##Navigate panes
; to move last active pane
o to move to the next pane
